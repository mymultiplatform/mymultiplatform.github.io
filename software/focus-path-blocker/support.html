Quick startup checks

  • Confirm the launch agent is running: launchctl list | grep com.focus.pathblocker.guard
  • If prompted by macOS for browser automation control, click Allow.
  • After a restart, allow 2-4 seconds for rules to apply in active tabs.

Current default rule set

  • YouTube Shorts routes are redirected.
  • Instagram reel/reels routes are redirected.
  • In Google Chrome, Instagram is restricted to /direct routes only.
